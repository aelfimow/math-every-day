\section{Exercise 69.3}

Given:
\[f(t) = e^{3t}\sin{4t}\]

Solution:
\[u(t) = e^{3t}\]
\[u'(t) = 3e^{3t}\]

\[v(t) = \sin{4t}\]
\[v'(t) = 4\cos{4t}\]

\[f'(t) = e^{3t}4\cos{4t} + \sin{4t}3e^{3t} =
          4e^{3t}\cos{4t} + 3e^{3t}\sin{4t} =
          e^{3t}(4\cos{4t} + 3\sin{4t})\]

Result:
\[f'(t) = e^{3t}(4\cos{4t} + 3\sin{4t})\]
