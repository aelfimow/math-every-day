\section{Exercise 68.1a}

Given:
\[f(x) = 5x^{2}\]

Using:
\[f(x) = ax^{n}\]
\[f'(x) = anx^{n - 1}\]

Solution:
\[f'(x) = 5\cdot2\cdot x^{2 - 1} = 10x^{1} = 10x\]

Result:
\[f'(x) = 10x\]

\section{Exercise 68.2b}

Given:
\[f(x) = 2,4x^{3,5}\]

Using:
\[f(x) = ax^{n}\]
\[f'(x) = anx^{n - 1}\]

Solution:
\[f'(x) = 2,4\cdot3,5\cdot x^{3,5 - 1} = 8,4x^{2,5}\]

Result:
\[f'(x) = 8,4x^{2,5}\]

\section{Exercise 68.2c}

Given:
\[f(x) = \frac{1}{x}\]

Using:
\[f(x) = ax^{n}\]
\[f'(x) = anx^{n - 1}\]

Solution:
\[f(x) = \frac{1}{x} = x^{-1}\]

\[f'(x) = -1\cdot x^{-1 - 1} = x^{-2} = -\frac{1}{x^{2}}\]

Result:
\[f'(x) = -\frac{1}{x^{2}}\]
