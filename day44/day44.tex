\section{Exercise 36.6}

Given:
\[\lim_{t\to 1}{\frac{\ln{t}}{t^{2} - 1}}\]

Solution:
\[u(t) = \ln{t}\]
\[u(1) = \ln{1} = 0\]

\[v(t) = t^{2} - 1\]
\[v(1) = 1^{2} - 1 = 1 - 1 = 0\]

\[u'(t) = \frac{1}{t}\]
\[u'(1) = \frac{1}{1} = 1\]

\[v'(t) = 2t^{2 - 1} = 2t\]
\[v'(1) = 2\cdot 1 = 2\]

\[\lim_{t\to 1}{\frac{\ln{t}}{t^{2} - 1}} = \lim_{t\to 1}{\frac{u'(t)}{v'(t)}} = \frac{1}{2}\]

Result:
\[\lim_{t\to 1}{\frac{\ln{t}}{t^{2} - 1}} = \frac{1}{2}\]
