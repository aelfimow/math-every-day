\section{$\cos{jx}$}

Given:
\[\cos{x} = \frac{1}{2}(e^{jx} + e^{-jx})\]

Solution:
\[\cos{jx} = \frac{1}{2}(e^{j(jx)} + e^{-j(jx)}) =
             \frac{1}{2}(e^{jjx} + e^{-jjx})\]

\[\cos{jx} = \frac{1}{2}(e^{-1\cdot x} + e^{-(-1)x}) =
             \frac{1}{2}(e^{-x} + e^{x}) = \frac{1}{2}(e^{x} + e^{-x})\]

\[\cosh{x} = \frac{1}{2}(e^{x} + e^{-x}) = \cos{jx}\]

Result:
\[\cos{jx} = \cosh{x}\]
