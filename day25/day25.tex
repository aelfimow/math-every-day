\section{Exercise 68.8}

Given:
\[y(x) = 5x^{2} - 3x + 1\]
\[y'(x_{0}) = 2\]

Solution:
\[y'(x) = 5\cdot2 x^{2 - 1} - 3\cdot x^{1 - 1} = 10x - 3\]
\[10x - 3 = 2\]
\[10x = 2 + 3 = 5\]
\[x = \frac{5}{10} = \frac{1}{2}\]

\[y(\frac{1}{2}) = 5(\frac{1}{2})^{2} - 3(\frac{1}{2}) + 1 =
                   \frac{5}{4} - \fraq{3}{2} + 1\]
\[y(\frac{1}{2}) = \frac{5}{4} - \fraq{1}{2} = \frac{5 - 2\cdot1}{4} = \frac{3}{4}\]

Result:
The point is $(\frac{1}{2}, \frac{3}{4})$.
