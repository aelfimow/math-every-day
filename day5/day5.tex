\section{Exercise 66.1}

\[f(t) = \sin{7t}\cos{2t}\]

Using:
\[\sin{\alpha}\cos{\beta} = \frac{1}{2}(\sin{(\alpha + \beta)} + \sin{(\alpha - \beta)})\]

\[f(t) = \frac{1}{2}(\sin{(7t + 2t)} + \sin{(7t - 2t)})\]
\[f(t) = \frac{1}{2}(\sin{9t} + \sin{5t})\]

Solution:
\[f(t) = \sin{7t}\cos{2t} = \frac{1}{2}(\sin{9t} + \sin{5t})\]
