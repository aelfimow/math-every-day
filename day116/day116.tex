\section{Exercise 1045a}

Given:
\[
x^{2} + xy + y^{2} = 6
\]

Solution:
\[
2x + y + xy' + 2yy' = 0
\]
\[
2x + y + y'(x + 2y) = 0
\]
\[
y'(x + 2y) = -(2x + y)
\]
\[
y' = -\frac{2x + y}{x + 2y}
\]

Result:
\[
y' = -\frac{2x + y}{x + 2y}
\]
