\section{Exercise 1}

Given:
\[f(x) = \tan(x)\]

Solution:
\[f(x) = \tan(x) = \frac{\sin{x}}{\cos{x}}\]

\[u(x) = \sin{x}\]
\[u'(x) = \cos{x}\]

\[v(x) = \cos{x}\]
\[v'(x) = -\sin{x}\]

\[f'(x) = \frac{\cos{x}\cos{x} - (-\sin{x})\sin{x}}{\cos^{2}{x}} =
         \frac{\cos^{2}{x} + \sin^{2}{x}}{\cos^{2}{x}} = \frac{1}{\cos^{2}{x}}\]

\[f'(x) = \frac{1}{\cos^{2}{x}} = \sec^{2}{x}\]


Result:
\[f'(x) = \frac{1}{\cos^{2}{x}} = \sec^{2}{x}\]
