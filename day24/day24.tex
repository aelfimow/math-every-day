\section{Exercise 68.7}

Given:
\[y(t) = 2t^{4} + 3t^{3} - t + 4\]
\[P_{1} = (0, 4)\]
\[P_{2} = (1, 8)\]

Solution:
\[y'(t) = 2\cdot4\cdot t^{4 - 1} + 3\cdot3\cdot t^{3 - 1} - 1\cdot t^{1 - 1} + 0\]
\[y'(t) = 8t^{3} + 9t^{2} - 1\]

Using $P_{1}$:
\[y'(0) = 8\cdot0 + 9\cdot0 - 1 = -1\]

Using $P_{2}$:
\[y'(1) = 8\cdot1^{3} + 9\cdot1^{2} - 1 = 16\]

Result:
\[y'(0) = -1\]
\[y'(1) = 16\]
