\section{Exercise 67.1}

\[f(x) = \sin{3x} + \sin{x}\]

Using: $\sin{\alpha} + \sin{\beta} = 2\sin{\frac{\alpha + \beta}{2}}\cos{\frac{\alpha - \beta}{2}}$

\[f(x) = 2\sin{\frac{3x + x}{2}}\cos{\frac{3x - x}{2}}\]
\[f(x) = 2\sin{\frac{4x}{2}}\cos{\frac{2x}{2}}\]
\[f(x) = 2\sin{2x}\cos{x}\]

Solution:
\[f(x) = \sin{3x} + \sin{x} = 2\sin{2x}\cos{x}\]
