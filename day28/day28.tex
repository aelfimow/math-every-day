\section{Exercise 69.1}

Given:
\[f(x) = 2x^{3}\cos{3x}\]

Solution:
\[u(x) = 2x^{3}\]
\[u'(x) = 2\cdot3\cdot x^{3 - 1} = 6x^{2}\]

\[v(x) = \cos{3x}\]
\[v'(x) = 3(-\sin{3x}) = -3\sin{3x}\]

\[f'(x) = 2x^{3}(-3\sin{3x}) + 6x^{2}\cos{3x} = 6x^{2}\cos{3x} - 6x^{3}\sin{3x}\]
\[f'(x) = 6x^{2}(\cos{3x} - 2\sin{3x})\]

Result:
\[f'(x) = 6x^{2}(\cos{3x} - 2\sin{3x})\]
