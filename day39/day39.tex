\section{Exercise}
\[f(x) = \cot(x)\]

Given:
\[f(x) = \frac{\cos{x}}{\sin{x}}\]

Solution:
\[u(x) = \cos{x}\]
\[u'(x) = -\sin{x}\]

\[v(x) = \sin{x}\]
\[v'(x) = \cos{x}\]

\[f'(x) = \frac{\sin{x}(-\sin{x}) - \cos{x}\cos{x}}{\sin^{2}{x}}\]
\[f'(x) = \frac{-\sin^{2}{x} - \cos^{2}{x}}{\sin^{2}{x}}\]
\[f'(x) = \frac{-(\sin^{2}{x} + \cos^{2}{x})}{\sin^{2}{x}}\]
\[f'(x) = \frac{-1}{\sin^{2}{x}} = -\csc^{2}{x}\]

Result:
\[f'(x) = -\frac{1}{\sin^{2}{x}} = -\csc^{2}{x}\]
