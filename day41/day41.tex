\section{Exercise 36.2}

Given:
\[\lim_{x\to 0}{\frac{\sin{x}}{x}}\]

Solution:
\[u(x) = \sin{x}\]
\[u(0) = \sin{0} = 0\]

\[v(x) = x\]
\[v(0) = 0\]

\[u'(x) = \cos{x}\]
\[u'(0) = \cos{0} = 1\]

\[v'(x) = 1\]
\[v'(0) = 1\]

\[\lim_{x\to 0}{\frac{\sin{x}}{x}} = \lim_{x\to 0}{\frac{u'(x)}{v'(x)}} = \frac{1}{1} = 1\]

Result:
\[\lim_{x\to 0}{\frac{\sin{x}}{x}} = 1\]
