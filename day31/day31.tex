\section{Exercise 69.4}

Given:
\[f(\theta) = e^{4\theta}\ln{3\theta}\]

Solution:
\[u(\theta) = e^{4\theta}\]
\[u'(\theta) = 4e^{4\theta}\]

\[v(\theta) = \ln{3\theta}\]
\[v'(\theta) = \frac{1}{\theta}\]

\[f'(\theta) = e^{4\theta}\frac{1}{\theta} + 4e^{4\theta}\ln{3\theta} =
               e^{4\theta}(\frac{1}{\theta} + 4\ln{3\theta})\]

Result:
\[f'(\theta) = e^{4\theta}(\frac{1}{\theta} + 4\ln{3\theta})\]
