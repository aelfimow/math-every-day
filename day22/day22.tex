\section{Exercise 68.5a}

Given:
\[f(x) = -4\cos{2x}\]

Solution:
\[f'(x) = -4\cdot2\cdot(-\sin{2x}) = -8(-\sin{2x}) = 8\sin{2x}\]

Result:
\[f'(x) = 8\sin{2x}\]

\section{Exercise 68.5b}

Given:
\[f(x) = 2e^{6x}\]

Solution:
\[f'(x) = 2\cdot6\cdot e^{6x} = 12e^{6x}\]

Result:
\[f'(x) = 12e^{6x}\]

\section{Exercise 68.5c}

Given:
\[f(x) = \frac{3}{e^{5x}}\]

Solution:
\[f(x) = \frac{3}{e^{5x}} = 3e^{-5x}\]

\[f'(x) = 3\cdot(-5)\cdot e^{-5x} = -15e^{-5x} = -\frac{15}{e^{5x}}\]

Result:
\[f'(x) = -\frac{15}{e^{5x}}\]
