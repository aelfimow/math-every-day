\section{Exercise 69.6}

Given:
\[i(t) = 15t\sin{3t}\]
\[t_{0} = 0,1\]

Solution:
\[u(t) = 15t\]
\[u'(t) = 15\]

\[v(t) = \sin{3t}\]
\[v'(t) = 3\cos{3t}\]

\[i'(t) = u(t)v'(t) + v(t)u'(t) = 15t3\cos{3t} + \sin{3t}15 = 15(3t\cos{3t} + \sin{3t})\]

\[i'(t_{0}) = 15(3\cdot 0.1\cos{3\cdot 0,1} + \sin{3 \cdot 0,1}) =
              15(0.3\cos{0,3} + \sin{0,3}) \approx 8,732\]

Result:
\[i'(t_{0}) \approx 8,732\]
